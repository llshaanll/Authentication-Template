Module: services/ApiClient

API Client Service Centralized HTTP client built on Axios for all API communications. Implements singleton pattern, request/response interceptors, automatic token injection, error handling, and authentication state management. Features: - Singleton instance (single HTTP client across app) - Automatic JWT token injection in request headers - Request/response logging for debugging - Global error handling with 401 redirect - CORS credentials support for cookies - Configurable timeout and base URL - Request/response interceptors - Centralized API configuration Architecture Pattern: Singleton + Interceptor Pattern - Single client instance shared across application - Interceptors for cross-cutting concerns - Separation from business logic - Reusable HTTP methods Security Features: - Automatic token injection (Authorization header) - Token validation on 401 responses - Automatic logout on authentication failure - HTTPS-only in production - CORS credentials for secure cookies Error Handling: - Global 401 handling (token expired/invalid) - Automatic redirect to login - Token cleanup on authentication failure - Error logging for debugging - Promise rejection for component handling Performance Considerations: - Single Axios instance (connection pooling) - 10-second timeout to prevent hanging requests - Efficient header management - Minimal overhead in interceptors
