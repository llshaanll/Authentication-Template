AuthService Class Singleton service that encapsulates all authentication-related operations. Provides a clean, consistent API for user registration, login, logout, and token management. Design Pattern: Singleton - Single instance shared across application - Consistent state management - Centralized configuration Dependencies: - ApiClient: HTTP communication layer - TokenManager: Token storage and retrieval State Management: - Stateless service (state stored in TokenManager/localStorage) - No internal state to maintain - Each method is independent
